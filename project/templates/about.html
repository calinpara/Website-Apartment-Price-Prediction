About this project

This website hosts a machine learning algorithm that aims to predict the future price of an apartment in Bucharest. There are 3 main parts to this project.

1. Scraping of Data

The 1st part consists of a script used for collecting data from a popular real estate website in Romania. I used Python together with the the Beautiful Soup web scraping library in order to gather features of Bucharest's apartments. All extracted records are stored inside of a csv file.

2. Analysis of Collected Data

After the creation of the css file and a quick glane at the records, I concluded that they were not appropriate for a machine learning model just yet. Thus began the analysis and cleaning of the data, so that the algorithm would turn out as accurate as possbile. To achieve this, I used Python and data science libraries. The image on the left shows a map of the location of the apartments gathered. In the end, a machine learning regression model was created, and was used in order to make predictions of future prices of apartments. For further information on this step, visit my Github page.

3. Website

The website you are reading this text on is made with the help of the Python, Flask library, SQLAlchemy. In order to make predictions, the user must create an account and login. After successfully signing in, the PREDICT page will be available for use.
